# 📝 DevNotes

Um app simples para criar, editar e organizar suas notas direto no navegador.

# ✨ Funcionalidades

* 📝 Criar notas
* ✏️ Editar em tempo real
* 💾 Salvamento automático (LocalStorage)
* 📌 Fixar notas
* 🔍 Buscar notas
* 🌙 Modo escuro/claro
* 📤 Exportar notas

# 🛠️ Tecnologias

* HTML
* CSS
* JavaScript

# ▶️ Como usar

1. Baixe os arquivos
2. Abra o `index.html` no navegador

Pronto, já pode usar 👍
